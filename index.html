Introdução ao HTML

Elementos do head

O elemento <head> contém metadados sobre o documento.

Título da página

O título da página é definido pelo elemento <title>.

Metadados

Os metadados são dados que descrevem outros dados.
Os metadados são definidos pelo elemento <meta>.

Estilos

Os estilos são definidos pelo elemento <style>.

Scripts

Os scripts são definidos pelo elemento <script>.

Elementos do body

O elemento <body> contém o conteúdo do documento.

Parágrafos

Os parágrafos são definidos pelo elemento <p>.

Listas

Ordenada

A lista ordenada é definida pelo elemento <ol>.

Não Ordenada

A lista não ordenada é definida pelo elemento <ul>.

Ligações

As ligações são usadas para navegar entre páginas da web.
As ligações são definidas pelo elemento <a>.

Dica

Em caso de dúvidas, pesquisar no site MDN Web Docs e no Can I Use.
